
# このリポジトリについて

[東京大学大学院理学系研究科 物理学専攻](https://www.phys.s.u-tokyo.ac.jp/)（東大理物）の院試問題に対する解答案をまとめているリポジトリです。

## ディレクトリ構造

年度ごとにディレクトリが分かれています。各ディレクトリの構成は以下のとおりです。

```
Science_Phys/
├── YYYY/                        # 各年度のディレクトリ（試験実施年）
│   ├── butsuriHXX.pdf           # 物理の問題（原本、年度によりファイル名が異なる）
│   ├── sugakuHXX.pdf            # 数学の問題（原本、年度によりファイル名が異なる）
│   ├── sp_YYYY.tex              # 全問まとめたメインLaTeXファイル
│   ├── content/
│   │   ├── PhysXX.tex           # 各問の解答LaTeXソース
│   │   └── out/
│   │       └── PhysXX.pdf       # 各問の解答PDF
│   └── out/
│       └── sp_YYYY.pdf          # 全問まとめた解答PDF
└── README.md
```

> **注意:** 年度によってファイル名・ディレクトリ構造が異なる場合があります。
> - 2009〜2012年度は `YYYY/content/Phys01/sub.tex` 形式、コンパイル済み PDF は `YYYY/out/master.pdf` です。
> - 2021〜2023年度は原本が `RXmaster.pdf` （令和X年の問題冊子）1ファイルのみで、数学・物理の別ファイルはありません。
